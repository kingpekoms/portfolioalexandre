Detail

Languages : PHP, HTML, CSS, Javascript, SQL.
Framework : Symfony.
Librairie : Bootstrap.
API : #.
Outils : Visual studio code, Virtual Box, SQL workbench, Teams, Github.


Contexte donné : Développement d'un site e-commerce de produits sportifs, permettant la notation et la recherche de ces mêmes produits. Les utilisateurs peuvent avoir plusieurs rôles, notamment celui de recevoir des alertes sur les stocks ou de gérer ces produits.
